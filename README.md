# New Aadhaar Details Update Flow (VisualVortex)

## Overview
This project is a high-fidelity HTML/JS prototype of the new Aadhaar address update flow. It demonstrates a streamlined, user-friendly interface adhering to "Government of India" aesthetic guidelines (Official Blue/Saffron/Green) while incorporating modern UX principles.

## Getting Started
To view the prototype, simply open the `index.html` file in any modern web browser.

No build step is required. The project uses:
- **Tailwind CSS** (via CDN) for styling.
- **Vanilla JavaScript** for interaction logic.
- **Google Fonts** & **Material Symbols** (CDN).

## Project Structure
- `index.html`: The main landing page / portal entry.
- `update_aadhaar_login.html`: Login screen with Head of Family options.
- `otp_verification.html`: Security check simulation.
- `dashboard.html`: The user dashboard after login.
- `address_entry.html`: Smart form for address details.
- `document_upload.html`: Document proof upload interface.
- `review_confirm.html`: Summary screen before submission.
- `payment.html`: Payment gateway simulation.
- `success.html`: Completion status and tracking.
- `assets/`: Contains images and resources.
- `demo_guide.js`: Injected logic for presentation notes (Press 'H' to toggle).

## Key Features
- **Unified Design System**: Consistent typography and color palette aligned with UIDAI standards.
- **Demo Mode**: Includes a presenter sidebar to explain design decisions.
- **Responsive**: Fully optimized for mobile and desktop views.
